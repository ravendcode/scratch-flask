# Simple Flask App From Scratch

### Features
1. Auth
2. Forms
3. SQLAlchemy
4. I18n
5. Heroku

### Usage
1. Edit `config.py`
2. `pip install -r requirements.txt`
2. `python init_db.py`
3. `npm i -g nodemon`
4. `npm i`
5. `npm run trans`
6. `npm run trans_en`
7. `npm run trans_ru`
8. `npm run trans_compile`
9. `npm run dev`
