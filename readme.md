# Simple Flask App From Scratch

### Features
1. Auth
2. Forms
3. SQLAlchemy
4. I18n
5. Heroku

### Usage
1. Edit `config.py`
2. `pip install -r requirements.txt`
3. `python manage.py create:db`
4. `python manage.py trans` or custom language `python manage.py trans -l en,ru`
5. `npm i -g nodemon`
6. `npm i`
7. `npm run dev`
